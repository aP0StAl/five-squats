# Five Squats

Five Squats помогает разложить дневную норму приседаний на маленькие подходы. По умолчанию приложение считает план с 08:00 до 22:00: по 5 приседаний в час, 70 за день.

Идея простая: не ловить каждый час будильник, а спокойно делать по 5-10 приседаний в течение дня и видеть, как ты идешь к вечернему плану.

## Возможности

- показывает, сколько приседаний стоит сделать к текущему часу;
- считает сделанное сегодня и остаток до дневного плана;
- добавляет подходы кнопками `+5` и `+10`;
- отменяет последний подход за сегодня;
- хранит данные локально в браузере;
- показывает историю за последние 14 дней;
- экспортирует и импортирует JSON;
- устанавливается на iPhone через Safari как обычная иконка на экране.

## Локальный запуск

Можно открыть `index.html` напрямую в браузере. Для проверки PWA и service worker лучше открыть через локальный сервер:

```sh
python3 -m http.server 8000
```

После этого страница будет доступна по адресу `http://localhost:8000`.

## Публикация на GitHub Pages

1. Создай репозиторий на GitHub, например `five-squats`.
2. Отправь туда эти файлы.
3. Открой `Settings -> Pages`.
4. В `Build and deployment` выбери `Deploy from a branch`.
5. Выбери ветку `main` и папку `/root`.
6. Открой опубликованный URL в Safari на iPhone.
7. Нажми `Share -> Add to Home Screen`.

Данные хранятся на устройстве, где открыта страница. Для переноса на другой браузер или телефон используй экспорт и импорт JSON.
